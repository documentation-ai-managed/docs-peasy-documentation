## What is Peasy?

Peasy provides a centralized space to organize and manage all your project documentation. You create dedicated documentation sites for APIs, SDKs, user guides, and internal wikis. Customize themes, add interactive components, and collaborate with your team in real-time.

Peasy handles hosting, search, analytics, and versioning, so you focus on content. Start with Markdown or MDX files and build professional docs without frontend expertise.

## Platform Setup Examples

Install Peasy CLI to sync local changes.

<CodeGroup tabs="npm,yarn,pnpm">
  ```bash
  npm install -g @peasy/cli
  peasy init my-docs
  ```
  ```bash
  yarn global add @peasy/cli
  yarn peasy init my-docs
  ```
  ```bash
  pnpm add -g @peasy/cli
  pnpm peasy init my-docs
  ```
</CodeGroup>

## Brand Guidelines and Customization

Follow these guidelines to match Peasy's brand.

<Tabs>
  <Tab title="Primary Color" icon="circle">
    Use `#3B82F6` for buttons and accents. Avoid shades darker than `#1E40AF`.
  </Tab>
  <Tab title="Typography" icon="type">
    Headings use Inter font at 24px (H2), 20px (H3). Body text at 16px.
  </Tab>
  <Tab title="Config Example" icon="code">
    
```javascript
// peasy.config.js
module.exports = {
  theme: {
    colors: {
      primary: '#3B82F6',
    },
    fonts: {
      heading: 'Inter, sans-serif',
    },
  },
};
```
    
  </Tab>
</Tabs>

<Expandable title="Advanced Custom CSS" default-open="false">
  Override styles with custom CSS in your config.

```css
:root {
  --peasy-primary: #3B82F6;
}

.button {
  background: var(--peasy-primary);
}
```
</Expandable>
